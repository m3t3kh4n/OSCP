
# enum4linux
- `-u`: username to use (default: "")
- `-p`: password to use (default: "")
- `-k`: user that exist on the remote system (default: administrator,guest,krbtgt,domain admins,root,bin,none)
- `-U`: get user list
- `-M`: get machine list
- `-S`: get share list
- `-a`: all
- `-r`: user enumeration via RID cycling
